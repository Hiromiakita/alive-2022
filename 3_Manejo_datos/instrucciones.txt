Utilizando el archivo datos.js

Vamos a hacer las siguientes peticiones desde el FRONTEND

Mostrar nombre y apellido de todos los alumnos 
http://localhost:3000/alumnos

Mostrar el promedio de un alumno, por ejemplo, para mostrar el promedio de Bell
http://localhost:3000/promedio/alumno/Bell
